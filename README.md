## Overview

Breast cancer is not a single disease. The **PAM50 intrinsic subtype classifier** partitions tumours into five molecularly distinct groups — each with different biology, treatment response, and survival outcomes. This project builds a complete clinical bioinformatics pipeline on synthetic TCGA-BRCA data, combining machine learning classification with survival analysis and SHAP-based interpretability.

```
Gene Expression (PAM50 + extra genes)
        │
        ▼
┌───────────────────┐     ┌───────────────────┐
│   XGBoost (400T)  │     │  Random Forest    │
│   5-fold CV       │     │  (500 trees)      │
└────────┬──────────┘     └─────────┬─────────┘
         │                          │
         ▼                          ▼
    PAM50 Subtype Predictions   SHAP Values
         │                          │
         ▼                          ▼
  Kaplan-Meier Curves    Gene-level biological
  Cox PH Regression      interpretation
```

## Biological Background

| Subtype | Molecular Features | Prognosis | Key Genes |
|---------|-------------------|-----------|-----------|
| **Luminal A** | ER+/PR+, HER2−, Ki-67 low | Best | ESR1, PGR, FOXA1, BCL2 |
| **Luminal B** | ER+/PR+, Ki-67 high | Intermediate | ESR1, MKI67, CCNB1 |
| **HER2-enriched** | HER2 amplified | Intermediate–poor | ERBB2, GRB7, CDH3 |
| **Basal-like** | ER−/PR−/HER2−, TP53 mut | Worst | KRT5, KRT14, EGFR, FOXC1 |
| **Normal-like** | Resembles normal breast | Good | BCL2, BAG1, SFRP1 |

*Sources: Parker et al. (2009) J Clin Oncol; TCGA Network (2012) Nature; Prat & Perou (2011)*

## Quick Start

### 1. Clone and install

```bash
git clone https://github.com/<your-username>/tcga-brca-pam50-survival.git
cd tcga-brca-pam50-survival
pip install -r requirements.txt
```

### 2. Run the full pipeline

```bash
python pipeline.py
```

Optional flags:
```bash
python pipeline.py --seed 123          # Different random seed
python pipeline.py --skip-shap         # Skip SHAP (faster, ~4s vs 11s)
```

### 3. Open the notebook

```bash
jupyter notebook notebooks/PAM50_Classification_Survival_Analysis.ipynb
```

## Data

### Synthetic Data (default)

Generated by `src/generate_data.py` using published PAM50 subtype profiles:
- **600 samples** across 5 subtypes (TCGA-proportioned)
- **60 genes**: 50 PAM50 canonical genes + 10 clinically relevant extras
- **Clinical covariates**: age, tumour stage, node status, tumour size
- **Survival data**: overall survival (months) + event indicator, Weibull-distributed

### Real TCGA-BRCA Data

To run on real data:

1. Download from [GDC Data Portal](https://portal.gdc.cancer.gov/projects/TCGA-BRCA)
2. Or use the `TCGAbiolinks` R package:
   ```R
   library(TCGAbiolinks)
   query <- GDCquery(project="TCGA-BRCA",
                     data.category="Transcriptome Profiling",
                     data.type="Gene Expression Quantification")
   GDCdownload(query)
   data <- GDCprepare(query)
   ```
3. Extract PAM50 gene columns and format as `synthetic_tcga_brca.csv`
4. Run `python pipeline.py`

## Methods

### Feature Engineering
- **Expression features**: log2-normalised transcript abundance (PAM50 genes + extras)
- **Proliferation score**: mean of MKI67, CCNB1, BIRC5, MELK, UBE2C, CDC20 (z-scored)
- **ER pathway score**: mean of ESR1, PGR, FOXA1, MLPH, NAT1, GATA3 (z-scored)

### Classification
- **XGBoost**: 400 estimators, max_depth=5, lr=0.08, colsample_bytree=0.75
- **Random Forest**: 500 trees, max_features=sqrt, balanced class weights
- **Validation**: 5-fold stratified cross-validation on training set

### Survival Analysis
- **Kaplan-Meier**: non-parametric survival function estimator, 95% CI via Greenwood's formula
- **Log-rank test**: pairwise comparison of subtype survival curves
- **Cox PH**: penalised (L2, λ=0.05) proportional hazards, Breslow baseline
- **Concordance index**: Harrell's C-statistic

### Interpretability
- **SHAP TreeExplainer**: exact (polynomial) Shapley values for tree-based models
- **Global importance**: mean |SHAP| across all samples and classes
- **Per-class SHAP**: directional gene contributions per subtype prediction
